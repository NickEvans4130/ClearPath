JetBrains Mono font files required:

  JetBrainsMono-Regular.ttf
  JetBrainsMono-Bold.ttf

Download from: https://www.jetbrains.com/lp/mono/
or: https://github.com/JetBrains/JetBrainsMono/releases

Copy the TTF files into this directory (assets/fonts/).
Also copy them into app/src/main/res/font/ as:
  jetbrains_mono_regular.ttf
  jetbrains_mono_bold.ttf

The font is referenced in Typography.kt via R.font.jetbrains_mono_regular
and R.font.jetbrains_mono_bold.
